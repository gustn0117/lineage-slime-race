# 마후 에이전트 (Lineage Classic Slime Race Agent)

리니지 클래식 슬라임 경주 화면을 감시해서 **라인업**과 **우승 결과**를
서버에 자동 전송하는 Windows 전용 프로그램.

## 빌드 (Windows PC, 최초 1회)

1. 이 `agent/` 폴더를 리니지를 돌리는 Windows PC에 복사.
2. `build.bat` 더블클릭.
   - Python 3.10 이상이 설치돼 있지 않으면 안내 메시지가 뜸.
     [python.org](https://www.python.org/downloads/) 에서 설치 → **"Add to PATH"** 체크.
   - 가상환경 생성 → 필수 패키지 설치 → PyInstaller로 단일 exe 빌드.
3. 완료되면 `dist\mahu-agent.exe` 파일이 생성됨. 이 파일만 있으면 됨.

## 실행

1. `mahu-agent.exe` 더블클릭.
2. 첫 실행이면 설정 창이 뜸. 아래 네 가지를 입력/지정:
   - **서버 URL** — 웹사이트 주소 (예: `https://lineage-slime-race.hsweb.pics`)
   - **에이전트 토큰** — 사이트 `/admin` → "에이전트 설정" 패널에서 복사
   - **레인 1~5 위치** — 게임 안에서 각 레인의 슬라임 **출발 지점**에 마우스를
     올려놓고 [현재 위치 저장] 버튼을 순서대로 클릭 (1번 레인 = 좌하단)
   - **대화창 영역** — 마후 메시지가 나오는 채팅 영역을 드래그
3. [설정 저장] → 자동으로 감시 시작. 콘솔에 이벤트 로그가 찍힘.
4. 닫기 = 종료. 다음부터는 저장된 설정으로 바로 시작됨.

## 동작

- 1~2초마다 5개 레인의 **노란 슬라임**을 색상 체크.
- 슬라임이 나타나면 → 각 레인으로 마우스 이동 → 툴팁(`#번호 이름`) OCR →
  라인업을 서버에 `POST /api/races/ingest` (type: lineup).
- 대화창을 주기적으로 OCR → `마후: 제 N회 게임의 우승자는 "#M 이름" 입니다!`
  메시지가 잡히면 우승자 정보를 서버에 `POST /api/races/ingest` (type: winner).

## 설정 다시 하기

- `dist` 폴더의 `config.json`을 지우고 `mahu-agent.exe`를 다시 실행.
- 또는 실행창에서 단축키 `Ctrl+C` 로 종료 후 `--setup` 플래그로 재실행.

## 문제 해결

- **"Python이 설치되어 있지 않습니다"** — python.org 에서 3.10 이상 설치
  (PATH 추가 필수).
- **라인업 OCR이 이상한 글자로 나옴** — 게임 해상도/자간에 따라 툴팁 폭이
  달라짐. 대화창 영역이 너무 좁거나 크지 않은지 재지정.
- **마후 메시지가 안 잡힘** — 대화창 영역이 마후 메시지가 나오는 줄을
  포함하는지 재확인.
- **마우스가 다른 창으로 움직임** — 게임이 최대화/창모드인지 확인. 창 위치
  바뀌면 레인 좌표 재설정 필요.
